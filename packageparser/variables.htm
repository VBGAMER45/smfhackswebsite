Variables used in SMF packages

There are several variables used in SMF packages. Some of the main ones are:

  • $boarddir - This is the directory SMF is installed in to (for example, /home/[username]/public_html/forum or /var/www/forum/)
  • $sourcedir - This is your SMF sources directory. On a default installation, this is $boarddir/Sources/
  • $themedir - This is the directory of the default theme (normally $boarddir/Themes/default)
  • $languagedir - This is the directory where all languages are situated. Normally $themedir/languages
